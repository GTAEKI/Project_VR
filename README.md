# Project_VR

## 👨🏻‍💻 제작 동기
기업 협약 프로젝트를 진행하기 이전에 프로그래밍팀 4명과 기획팀 4명이 합을 맞춰봤습니다.

## 🖥️ 프로젝트 소개
- 경일 게임아카데미 프로그래밍반 4명 / 기획반 4명 협업
- VR게임으로써 기획서에 맞춰 일정 관리 및 게임 제작
- 성벽을 지키는 게임입니다. 주인공은 마법을 사용하며 성벽을 향해 걸어오는 보스몬스터를 처치합니다.
- 보스몬스터는 운석과 졸개를 소환하여 성벽을 공격합니다.
